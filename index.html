👀黑白卡视觉追踪训练

发展能力：视觉发展、注意力、专注力

操作步骤：

  1. 准备黑白对比度高的卡片或棋布
  2. 将卡片放在宝宝眼前20-30cm处
  3. 在宝宝视野中央吸引注意力后，缓慢向左右移动（每次不超过半分钟）
  4. 2个月宝宝可追视至90度

📚 来源：知乎专栏《15个分年龄段家庭感统训练游戏》

💪俯卧抬头练习

发展能力：颈部力量、运动控制能力、肌肉发展

操作步骤：

  1. 让宝宝趴在干净的床或垫子上
  2. 在宝宝前方20-30cm处放置彩色玩具或镜子
  3. 鼓励宝宝抬头观看
  4. 每次练习2-3分钟，根据宝宝状态调整

📚 来源：沈阳市人民政府《疫情期间亲子互动促健康》科普

💬亲子对话与表情交流

发展能力：语言能力、情感发展、社交能力、安全感建立

操作步骤：

  1. 让宝宝躺在妈妈双腿上或小床上
  2. 将脸放在距离宝宝30cm左右的地方
  3. 与宝宝建立目光联系，用温柔的声音说话
  4. 边说话边左右缓慢移动脸部，让宝宝追踪
  5. 变换表情（微笑、张嘴等），观察宝宝反应

📚 来源：澎湃新闻《在家早教系列：0-1月宝宝的生长发育特点及游戏方式》

🔔手摇铃听觉训练

操作步骤：

  1. 在宝宝左侧或右侧轻轻摇响手摇铃
  2. 观察宝宝是否转头寻找声源
  3. 变换位置，从不同方向发出声音
  4. 使用不同音色的玩具，丰富听觉刺激

📚 来源：澎湃新闻《出生就能玩！十个0-6个月亲子互动小游戏》
